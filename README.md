## Development Setup
1. Follow the [Flutter install instructions](https://flutter.dev/docs/get-started/install) for your platform.
2. Optional: Install [nix](https://nixos.org/) with [flake support](https://nixos.wiki/wiki/Flakes#Installing_flakes) turned on if you want to mess with some of the Nix tooling. 
3. Optional: Install [direnv](https://direnv.net/) for a more convenient shell experience.
4. Then: 
```bash
# Either...
hub clone iridium-eve/iridium
# Or...
git clone git@github.com:iridium-eve/iridium.git

cd iridium

flutter pub get
flutter run
```
